# Assets

This file maps the project's evidence layer.

## Buckets

- `assets/sessions/` - session records and handoffs.
- `assets/plans/` - plans, options, and discarded approaches.
- `assets/research/` - research, source notes, and raw findings.

## Promotion

Promote durable conclusions into the owning live context file. Keep assets as evidence.
