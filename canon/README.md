# Canon

Canon holds ContextKit's foundational laws and context lenses.

It distills the thesis into durable rules that should constrain every shipped
surface: standards, bundled runtime, bundled guides, templates, validation,
CLI behavior, and demos.

Canon is not an implementation guide. It should stay high enough to apply
across mechanisms.
