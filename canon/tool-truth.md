# Tool Truth

Power: freshness.

Command-answerable knowledge belongs in the command.

Decision Question: can a live tool answer this more accurately at use time?

## Law

If a tool can answer a question through help, doctor, connections, guide,
contract, or another live surface, ContextKit routes to that surface instead of
copying the answer into always-on prose.

Readiness is proven at use time by the owning tool, not inferred from context
presence.

## Failure Signs

- README or runtime copies a CLI's full command table;
- context asserts a tool is ready without running its doctor;
- output schemas are transcribed into prose.
