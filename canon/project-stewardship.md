# Project Stewardship

Power: respect for existing work.

An existing project has standing. ContextKit enters it as a steward, not as an
owner of a blank surface.

Decision Question: does this change preserve the integrity of existing work
before reshaping it?

## Law

Before ContextKit changes a project, it inspects, preserves existing material
unless replacement is justified, makes consequential structural change
reviewable, prefers recoverable change, blocks on ambiguous state, and
classifies existing doctrine before removing or superseding it.

The tool serves the project.

## Failure Signs

- existing work is replaced before its owner is understood;
- ambiguous state is silently normalized;
- human-authored project memory is treated as disposable scaffolding.
